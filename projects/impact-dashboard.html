Community Impact Dashboard

A full-stack web application designed to help organizations manage events, attendees, and donations through a centralized dashboard.

Tech Stack

  • Django & Django REST Framework
  • JavaScript, HTML, CSS
  • SQLite (development)

Key Features

  • RESTful APIs for events, attendees, and donations
  • Live dashboard with KPI metrics
  • Search and sorting functionality
  • Admin interface for secure data management

Challenges & Learnings

I focused on clean API design, separation of concerns between frontend and backend, and ensuring the frontend consumed real-time backend data correctly.

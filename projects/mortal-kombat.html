Mortal Kombat (Python)

A two-player arcade-style fighting game inspired by classic fighting games, built in Python with real-time input handling and animations.

Tech Stack

  • Python
  • Pygame (graphics, input handling, and game loop)

Key Features

  • Local two-player multiplayer combat
  • Keyboard-based movement, attacks, and blocking
  • Health bar system with win and loss conditions
  • Sprite-based character animations and backgrounds
  • Game reset and replay functionality

Challenges & Learnings

This project strengthened my understanding of real-time game loops, collision detection, animation timing, and managing game state across multiple interacting systems in Python.
